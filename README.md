# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Исследовательский анализ данных сервиса Яндекс Афиша для выявления факторов, влияющих на повторные покупки пользователей.

## Цель
Понять, какие характеристики первого заказа и поведения пользователя связаны с его возвращаемостью.

## Данные
Датасет содержит ~290 тыс. заказов за период июнь–октябрь 2024 года.

## Основные шаги анализа
- Загрузка данных из SQL
- Предобработка и очистка
- Построение профиля пользователя
- Анализ факторов возврата
- Корреляционный анализ

## Основные выводы
- 62% пользователей совершают повторный заказ
- Более частые покупки связаны с высокой лояльностью
- Размер и частота заказов важнее характеристик первого заказа

## Стек
Python, pandas, matplotlib, seaborn, phik, SQL
